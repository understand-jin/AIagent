# AI 에이전트 과제 기획서 (v2.0)

---

## 1. 과제 개요

- **과제명:** 대웅제약 맞춤형 AI 시장 기회 분석 & 실행 전략 도출 시스템
- **핵심 컨셉 (v2.0 고도화):** 단순한 '내 뉴스 찾기'를 넘어, **'시장 트렌드에서 대웅의 기회를 발굴'**하는 전략 에이전트 — 일반적인 산업 키워드(예: 비만치료제, AI 신약)를 검색하여 제약 전문가 시나리오를 바탕으로 대웅제약만의 차별화된 전략 로드맵을 생성
- **타겟 페르소나:** 시장 변화에 대응하여 신규 사업 기회를 발굴해야 하는 대웅제약 전략기획팀, R&D 기획 담당자, 신사업 부서 의사결정자

---

## 2. 에이전트 아키텍처 설계 (고도화 버전)

### 아키텍처 다이어그램

```
[사용자 입력: 시장 키워드]
          │
          ▼
┌─────────────────────────────────┐
│  노드 1: 수집가 (Researcher)     │
│  - Tavily API 기반 시장 뉴스 수집 │
│  - 대웅 특정 키워드에 국한되지 않음 │
│  - 최신 산업 동향 20건 정제      │
└────────────┬────────────────────┘
             │ Market News Data
             ▼
┌─────────────────────────────────┐
│  노드 2: 기회 분석가 (Analyst)   │
│  - 수집된 일반 뉴스 분석         │
│  - 대웅제약 입장의 3대 기회 발굴  │
│  - 대웅의 핵심 역량과 이슈 결합   │
└────────────┬────────────────────┘
             │ Opportunity Analysis
             ▼
┌─────────────────────────────────┐
│  노드 3: 전략가 (Strategist)     │
│  - 도출된 기회 -> 실행 전략 변환  │
│  - 3개월/1년 단위 구체적 로드맵   │
│  - 기대 효과 및 KPI 설정         │
└────────────┬────────────────────┘
             │
             ▼
     [Flask 웹 UI 전략 보고서 출력]
```

### 설계 의도 (고도화 포인트)

1.  **전략적 통찰력 강화**: 기존에는 대웅 관련 뉴스가 없으면 분석이 어려웠으나, v2.0에서는 **일반 시장 뉴스에서 대웅이 무엇을 해야 할지(What to do)**를 도출하도록 로직을 개편함.
2.  **데이터 품질 및 보안**: API 키 보안을 위해 `.env` 시스템을 도입하고, 환경 변수를 통해 안전한 배포 환경을 구축함.
3.  **사용자 경험(UX) 개선**: 파이프라인 시각화와 로딩 상태 메시지를 기획 의도에 맞게 직관적으로 변경.

### 노드 구성 (v2.0)

| # | 노드명 | 역할 / 페르소나 (고도화 내용) |
|---|--------|----------------|
| 1 | **수집가 (Researcher)** | 입력된 키워드에 집중하여 시장 전체의 최신 뉴스를 수집합니다. 정보의 폭을 넓히기 위해 '대웅' 검색어를 강제하지 않고 순수 시장 트렌드 데이터를 확보합니다. |
| 2 | **기회 분석가 (Analyst)** | **(신규 역할)** 수집된 시장 정보를 대웅제약의 전략기획팀 시각에서 재해석합니다. 대웅의 R&D 역량, 영업망을 고려하여 3대 기회 요인을 도출합니다. |
| 3 | **전략가 (Strategist)** | 도출된 기회를 실무에서 실행 가능한 액션 아이템으로 변환합니다. 단기(3개월) 및 중기(12개월) 로드맵을 포함한 전문 비즈니스 보고서를 생성합니다. |

## 3. 주요 개선 및 고도화 기록

### [Security] 보안 강화
- **API 키 분리**: 하드코딩된 Groq, Tavily API 키를 `.env` 파일로 분리.
- **Git 보안**: `.gitignore` 설정을 통해 중요 정보가 GitHub에 노출되지 않도록 전면 조치.

### [Logic] 분석 관점의 대전환
- **Search -> Insight**: '대웅 관련 뉴스 검색'에서 **'시장 이슈 기반 대웅 전략 제안'**으로 에이전트의 사고 방식을 고도화.
- **프롬프트 엔지니어링**: 제약 산업 특화 비즈니스 톤 앤 매너 적용, R&D 및 마케팅 관점의 분석 프레임워크 주입.

### [Deployment] 배포 최적화
- **Port 바인딩**: 외부 환경(Render 등)의 dynamic port를 지원하도록 `app.py` 수정.
- **Dependencies**: `python-dotenv`, `lxml`, `gunicorn` 등을 추가하여 프로덕션 레벨의 호환성 확보.
- **Procfile**: PaaS 환경 자동 배포를 위한 설정 파일 추가.

### [Infrastructure] 버전 관리
- **GitHub 연동**: 지정된 공식 저장소(`understand-jin/AIagent`)에 코드 및 문서 형상 관리 체계 구축.

## 4. 최종 결과물 구성

### 시스템 구성 파일

| 파일 | 역할 |
|------|------|
| `app.py` | Flask 메인 엔진. 3개 전략 노드 로직 탑재 |
| `.env` | **(신규)** API 키 및 환경 변수 보안 관리 |
| `.gitignore` | **(신규)** 보안 및 불필요 파일 업로드 방지 |
| `procfile` | **(신규)** 웹 서버 실행 정의 (배포용) |
| `link.py` | Tavily 뉴스 API 통신 및 환경 변수 로드 |
| `check.py` | BeautifulSoup 기반 뉴스 본문 크롤링 |
| `requirements.txt` | 시스템 라이브러리 명세 (배포용 패키지 포함) |

### 사용 기술 스택 (Production-Ready)

- **Language:** Python 3.11+
- **Framework:** Flask (Web), Groq (LLM - Llama 3.3 70B)
- **Data Source:** Tavily News API
- **Deployment:** Git/GitHub, Procfile-based Web Server
- **UI:** CSS Grid/Flexbox 기반 반응형 대시보드
